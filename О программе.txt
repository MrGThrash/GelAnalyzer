Программа позволяет анализировать полимерные микрогели, путём расчёта их основных параметров: 
1) числа отдельных молекул геля в растворе
2) радиуса инерции вращения в плоскости XY и по оси Z (как для одной молекулы, так и для группы молекул)
3) площади покрытия гелем границы фазы

Использован язык C#, среда разработки - Microsoft Visual Studio 2017 Community Edition. 
Для создания интерфейса использованы Windows Forms. 
Программа в качестве входных данных принимает файл с расширением .xyzr, в котором при моделировании микрогелей, например, 
в пакете lammps, записаны координаты "бидов" - частиц.
Выходные данные - вышеуказанные параметры микрогелей.

Основной исполняемый файл - Form1.cs. 
В файлах analyzer.cs, structformer.cs, moldata.cs описаны методы, которы используются при расчёте параметров микрогелей. 
Файл program.cs - основная точка входа программы.
В файле fileworker.cs описаны алгоритмы работы с файлами.
